
## 🛠️ Tecnologias Utilizadas

| Tecnologia | Finalidade |
|------------|------------|
| **React 18** | Framework para construção da interface |
| **SheetJS (XLSX)** | Geração de planilhas Excel estilizadas |
| **jsPDF + autoTable** | Geração de relatórios PDF profissionais |
| **JavaScript ES6+** | Lógica de negócio e helpers |

## 📦 Como Executar o Projeto

### Pré-requisitos

- Node.js (versão 14 ou superior)
- npm ou yarn

### Passo a passo

```bash
# 1. Clone o repositório
git clone https://github.com/SEU-USUARIO/os-service.git

# 2. Acesse a pasta do projeto
cd os-service

# 3. Instale as dependências
npm install

# 4. Execute o projeto
npm start
